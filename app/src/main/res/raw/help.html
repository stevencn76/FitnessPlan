How to create a new plan
In the main interface(Plan List), click "Create New Plan" button to show a new view which contains a date selector. Click the day which you want to choose, then click confirm button to create the plan. After confirming, the user interface will jump to plan list window.
How to delete a plan
In the main interface(Plan List), long press on one plan which you want to delete, then a popup menu would be shown with a delete menu item. Click the delete menu item, that plan would be deleted.
How to tick a exercise
Click the plan item in the main interface(Plan List), you will jump to the details for the plan. In the plan details window, click one day on which you want to tick some exercises. Jump to day exercises window in which you can tick the exercises which you have done. Also, you can click one exercise to show the details for it.
